Apps you can install

JellyfinMultiple Tizen builds (10.10.z, 10.11.z, SmartHub, OSA, GrayFix, OblongIcon and more) — picks the latest 5 releases automatically.
Jellyfin AVPlayAVPlay video player patches for better Samsung TV compatibility, including a SmartHub variant.
Jellyfin Legacy10.8.z build kept around for older Tizen models.
MoonfinJellyfin fork optimised for the viewing experience on Samsung Smart TVs.
LitefinLightweight Jellyfin client designed to run on legacy hardware.
Tizen Community PackMoonlight, Fireplace, TVApp, Twitch, Club Info Board, Doom, Transport Tycoon Deluxe, Nuvio…
Custom WGTBring your own .wgt — useful for builds you maintain yourself.

What it does for you

🔎 Auto TV discoveryScans your network and lists Samsung TVs ready to receive an install.
🔐 Tizen certificate handlingGenerates and manages the developer certificate, including the Samsung account login flow.
🧩 Plugin patchingOptional fixes such as the YouTube plugin patch baked right into the install step.
🎨 Jellyfin themesOne-click apply for popular community themes (Obsidian, Solaris, Nebula, Ember, Void).
👥 Multi-user configPush Jellyfin preferences for one or many users in a single shot.
🌍 Localised UIMultiple languages with full RTL support.

📺 Got an older Samsung TV?
If your TV runs Orsay OS (pre-2015 models), you'll need the Jellyfin Orsay Installer instead.
